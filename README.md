# 🍱 HakPlace CLI

터미널에서 빠르고 간편하게 학식 메뉴를 확인할 수 있는 CLI 도구입니다.  
<br>

## 📥 다운로드 및 설치 (일반 사용자)

파이썬 설치나 복잡한 설정 없이, 아래 링크에서 실행 파일 하나만 다운로드하면 사용할 수 있습니다.

> 👉 **[최신 릴리즈 다운로드 가기 (hakplace.exe)](https://github.com/WaltDev29/HakPlace-CLI/releases/latest)** 

다운로드 받은 `hakplace.exe` 파일을 원하시는 폴더에 두고, 해당 폴더를 환경 변수로 등록하면 됩니다.  
<br>

## 🚀 사용 방법

터미널(CMD 또는 PowerShell)을 열고 아래 명령어를 입력하세요.

### 1. 금일 메뉴 확인
오늘의 조식, 중식, 석식 메뉴와 평점을 확인합니다.
```bash
day
```

### 2. 금주 메뉴 확인
이번 주(월~금) 전체 식단 메뉴와 평점을 확인합니다.
```bash
week
```
