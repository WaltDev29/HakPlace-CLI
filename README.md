## 📥 다운로드 및 설치 (일반 사용자)
최신 릴리즈를 다운 받고, 해당 폴더를 환경 변수로 등록하면 됩니다. 
> 👉 **[최신 릴리즈](https://github.com/WaltDev29/HakPlace-CLI/releases/latest)**  
<br>

## 🚀 사용 방법

터미널(CMD 또는 PowerShell)을 열고 아래 명령어를 입력하세요.

### 1. 금일 메뉴 확인
오늘의 조식, 중식, 석식 메뉴와 평점을 확인합니다.
```bash
day
```

### 2. 금주 메뉴 확인
이번 주(월~금) 전체 식단 메뉴와 평점을 확인합니다.
```bash
week
```
